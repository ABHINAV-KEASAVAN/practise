Tables in HTML

Tables are used to represent the real life structural data.
It has a special attribute used to add border around the table called <"BORDER">. 
 
It has some other attributes called :
<"tr"> used to display table row.
<"td"> used to display table data.
<"th"> used to display table header. 

Below I have created a simple table using the above attributes.
